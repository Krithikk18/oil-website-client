# Vidhya's Oil - Website Client

## Project Structure
- `artifacts/vidhyas-oil/` — **Main website** (run this for local development)
- `artifacts/api-server/` — API server (Replit only)
- `artifacts/mockup-sandbox/` — Design sandbox (Replit only)
- `vidhyas-oil-standalone/` — Standalone copy for local dev without pnpm

## Running Locally
```bash
cd vidhyas-oil-standalone
npm install
npm run dev
```
Opens at http://localhost:5173
